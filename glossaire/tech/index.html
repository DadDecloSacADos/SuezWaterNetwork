Glossaire — Termes techniques¶

A¶

AG Grid Bibliothèque composant tableaux web. Utilisée dans AQWN pour affichage données (listes fuites, événements, capteurs).

Angular Framework JavaScript moderne. Utilisé dans prototype HTML/Angular remplaçant WPF.

API REST Interface programmation permettant requêtes/réponses HTTP. AQWN expose APIs pour intégrations externes.

B¶

Backend Système serveur traitant logique métier, BD, intégrations. Détails technologie manquants (TODO: .NET ?).

C¶

CWE-79 (XSS) Vulnérabilité injection cross-site. Détectée dans AQWN, on hold (pas de plan remédiation claire).

E¶

EPANET Voir glossaire métier.

H¶

HTML5 Norme web moderne. Utilisée dans prototype interface remplaçant WPF.

I¶

InfluxDB Base de données séries temporelles. Probable stockage données capteurs AQWN (confirmation manquante).

INP (EPANET file format) Format fichier texte propriétaire EPANET. Définit topologie réseau, paramètres hydrauliques.

J¶

JavaScript / TypeScript Langages scripting web. POC TypeScript referenced architecture en cours.

M¶

Middleware Couche logicielle intermédiaire traitant requêtes. Intégrations AQWN ↔ systèmes externes.

O¶

OpenLayers Voir glossaire métier.

P¶

PLTF Voir glossaire métier.

POC (Proof of Concept) Prototype exploratoire validant concept. Time Series architecture POC en cours.

Prometheus Alternative InfluxDB pour séries temporelles (probabilité plus basse).

R¶

REST API Voir API REST.

S¶

Sandboxing Isolation exécution code pour sécurité. Prendre en compte pour intégrations tierces.

SIG/GIS (Système d'Information Géographique) Système gestion données géographiques. Couches SIG affichées via OpenLayers.

SQL Langage requête bases de données. Probable pour BD métier AQWN (détails manquants).

T¶

Time Series Database BD spécialisée séries temporelles (InfluxDB, Prometheus, etc.). Stocke millions métriques capteurs.

TypeScript Superset JavaScript avec types statiques. POC referenced architecture.

V¶

Vector (données) Format géométrique (points, lignes, polygones) vs raster (pixels). OpenLayers utilise vecteurs.

W¶

WPF (Windows Presentation Foundation) Framework .NET UI bureautique. Interface legacy AQWN. Migration vers HTML/Angular en cours.

Web Service API exposée via HTTP (SOAP, REST). AQWN exposes services tiers.
