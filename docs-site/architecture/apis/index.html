APIs¶

REST API publique¶

Base URL : https://aqwn.suez.com/api/v1/

Endpoints fuites¶

GET    /leaks                    # Lister fuites
POST   /leaks                    # Créer fuite
GET    /leaks/{id}              # Détail fuite
PUT    /leaks/{id}              # Modifier fuite
DELETE /leaks/{id}              # Supprimer fuite

Endpoints événements¶

GET    /events                   # Lister événements
POST   /events                   # Créer événement
GET    /events/{id}             # Détail événement

Endpoints capteurs¶

GET    /sensors                  # Lister capteurs
POST   /sensors                  # Ajouter capteur
GET    /sensors/{id}/timeseries # Séries temporelles

Endpoints rapports¶

GET    /reports/iwa             # Bilan IWA
GET    /reports/daily           # Rapport journalier
POST   /reports/custom          # Rapport custom

Authentication¶

Authorization: Bearer <token>

Exemples requêtes¶

# Créer fuite
POST /leaks HTTP/1.1
{
  "type": "main",
  "location": {"x": 123.456, "y": 78.910},
  "dma": "DMA_01",
  "volume_estimated": 5.0,
  "volume_unit": "l/min"
}

# Lister événements zone
GET /events?dma=DMA_01&limit=10

Webhooks¶

Event subscriptions :

POST /webhooks
{
  "event": "leak.detected",
  "url": "https://external.system/webhook"
}
