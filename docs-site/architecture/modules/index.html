Modules techniques¶

Modules Core¶

EPANET Engine¶

  • Rôle : Simulation hydraulique réseau
  • Technologie : Open-source EPA
  • Intégration : Fichier INP, API, résultats séries temporelles
  • État : Production, remplace Ganessa

Data Correlator¶

  • Rôle : Détection événements + corrélation données
  • Technologie : Statistique, détection change point
  • Input : Séries temporelles capteurs, événements
  • Output : Corrélations identifiées, scores
  • État : Production

Time Series Architecture (POC)¶

  • Rôle : Stockage/requête efficace séries temporelles
  • Technologie : TypeScript referenced architecture
  • État : POC, exploration en cours

Modules Présentation¶

WPF Configurateur (Legacy)¶

  • Rôle : Interface configuration avancée
  • Technologie : Windows Presentation Foundation (.NET)
  • État : Maintenance, migration en cours
  • Futur : Remplacé par HTML/Angular prototype

HTML/Angular Prototype¶

  • Rôle : Nouvelle interface web configuration
  • Technologie : HTML5, Angular, TypeScript
  • État : POC/Prototype
  • Blocage : Convergence design manquante

Modules Données¶

PLTF Connectors¶

  • Rôle : Intégrations données externes (standardisé SUEZ)
  • État : Remplace progressivement connecteurs WN-spécifiques
  • Exemples : AMR, GIS, APIs tierces

OpenLayers Cartographie¶

  • Rôle : Affichage interactif réseau, capteurs, fuites
  • Technologie : Open-source web mapping
  • Versions : 8.x (actuelle), v10.3 (en upgrade)
  • État : Production, performance améliorée

Modules Métier¶

  • Hydraulic Simulator → module EPANET
  • Leak Detection → combos Correlator + LLI
  • Event Management → SEF engine
  • IWA Calculator → (bloqué, 48%)
  • Reporting Engine → KPI, export PDF/Excel
