Configuration & Administration¶

Objectif de cette page

Comprendre la gestion des entités, configuration et droits utilisateurs.

Entités configurables¶

Hiérarchie¶

Contrat (Utility)
  └─ Région / Service
     └─ DMA (District Metered Area)
        ├─ Zones hydrauliques
        ├─ Secteurs (SEF)
        └─ Capteurs & Équipements
           ├─ Débitmètres
           ├─ Manomètres
           ├─ Loggers acoustiques
           └─ Capteurs qualité

Propriétés configurables¶

Entité Config possibles
DMA Nom, limites géo, zone de pression
Zone Limites réseau, capteurs associés
Secteur Conduites incluses, vannes fermeture
Capteur Type, localisation, fréquence, seuils alerte
Modèle EPANET Fichier INP, mapping entités, scheduler
Seuils alertes Par capteur/zone, minima/maxima

Rôle Permissions
