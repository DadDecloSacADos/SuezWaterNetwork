Aquadvanced WaterNetworks — Portail de Documentation 360°¶

À retenir

Aquadvanced WaterNetworks (AQWN) est la solution de gestion intelligente des réseaux d'eau potable de SUEZ Digital Solutions. Ce portail est votre point d'entrée unique pour comprendre, opérer et faire évoluer le produit.

Qu'est-ce qu'Aquadvanced WaterNetworks ?¶

Aquadvanced WaterNetworks est une plateforme intégrée pour la gestion des réseaux d'eau potable, combinant :

  • Surveillance hydraulique en temps réel : Intégration du moteur EPANET pour la modélisation hydraulique et la simulation de scénarios
  • Détection et localisation de fuites : Corrélation automatique des données acoustiques pour identifier et localiser les fuites dans le réseau
  • Gestion des capteurs IoT : Collection et analyse des données provenant de capteurs de pression, débits, qualité de l'eau
  • Bilan hydraulique IWA : Calcul des indicateurs de performance selon le standard International Water Association
  • Gestion d'événements et alertes : Système SEF (Sector Event Flow) pour la détection automatique et la gestion des secteurs affectés
  • Reporting et tableaux de bord : KPIs, rapports de performance, visualisation des données
  • Cartographie interactive : Représentation géospatiale du réseau avec OpenLayers

AQWN s'adresse aux utility managers, opérateurs eau, ingénieurs eau et décideurs des services d'eau.

Risques critiques actuels¶

IWA Water Balance — BLOQUÉ

Complétion : 48% (la plus basse)

  • 55 User Stories ouvertes
  • Spécification incomplète (v4)
  • Impact : Impossibilité de calculer les indicateurs IWA A/B/C/D de façon fiable
  • Voir page dédiée →

CWE-79 XSS — On Hold

Vulnérabilité de sécurité détectée (injection XSS). Pas de plan de remédiation clair.

Migration WPF → HTML

Configurateur legacy WPF en maintenance parallèle avec prototype HTML/Angular. Double charge de travail.

Technologie principale¶

  • Moteur hydraulique : EPANET (remplace Ganessa) — simulation hydraulique complète
  • Cartographie : OpenLayers 10.3 — couches SIG, visualisation réseau temps réel
  • UI : WPF (héritage) + migration en cours vers HTML/Angular
  • Connecteurs : PLTF (remplacement progressif des connecteurs WN-spécifiques)
  • Séries temporelles : POC TypeScript referenced architecture
  • AG Grid : Tableaux de données interactifs
