Cartographie & SIG¶

Objectif de cette page

Comprendre la cartographie interactive, les couches SIG et la visualisation du réseau.

OpenLayers¶

OpenLayers : Bibliothèque open-source pour cartographie web.

Version en AQWN¶

  • Actuelle : 8.x
  • En upgrade : v10.3 (2024)
  • Nouvelles fonctionnalités : Performance, vecteurs améliorés, clustering

Couches SIG¶

Couche Source Éditable
Réseau (conduites) AQWN DB ✅
Nœuds (jonctions) AQWN DB ✅
Capteurs AQWN DB Readonly
Zones (DMA) AQWN DB ✅
Fuites AQWN DB ✅
Fonds (OSM, IGN) Tuiles externes Readonly

Styles thématiques¶

Vue Pression¶

  • Conduites colorées par pression (bleu = faible, rouge = élevée)
  • Nœuds avec icône de débit

Vue Débit¶

  • Conduites épaisseur = débit (conduite grosse = fort débit)
  • Légende graduée

Vue Fuites¶

  • Fuites = icônes spéciales
  • Couleur = état (apparition / détection / réparation)

Vue Qualité¶

  • Points de mesure avec symboles qualité
  • Zones colorées conformité chlore/turbidité

Intégrations¶

  • Hydraulique : Affichage résultats simulation EPANET
  • Détection fuites : Localisation LLI sur map
  • Événements : Zones affectées soulignées
